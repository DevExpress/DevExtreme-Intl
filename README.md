# [Deprecated] DevExtreme-Intl

This package is no longer maintained. Intl integration is now implemented in the [DevExtreme monorepo](https://github.com/DevExpress/DevExtreme/tree/HEAD/packages/devextreme/js/__internal/core/localization/intl).

To translate DevExtreme component UI elements to a different language, refer to the following user guide: [DevExtreme Localization](https://js.devexpress.com/Documentation/Guide/Common/Localization/).
